# PROYECTO JAVA - SENA

## Descripción
Sistema de gestión para bar/restaurante desarrollado en Java con funcionalidades de ventas, alertas, inventario y procesamiento con IA.

## Cómo ejecutar
1. Abrir carpeta en VS Code
2. Compilar: `javac *.java`
3. Ejecutar: `java Administrador`

## Clases principales
- `Venta.java` - Gestión de ventas
- `Alerta.java` - Sistema de alertas
- `IA_Processor.java` - Procesamiento inteligente
